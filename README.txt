****Instrucciones De Ejecución*****
****--------------------------*****
1. Descombrima el archivo .zip localizado en sicua.
2. Abra la linea de comandos y coloquese en el directorio donde va a descargar el repositorio.
3. Desde linea de comandos git clone https://github.com/krmarzola/TareaNo3.git sobre el directorio que ha seleccionado.
4. Ejecute en linea de comandos sobre el directorio creado donde se encuentra el archivo poom.xml: mvn clean.
4. Ejecute en linea de comandos mvn compile.
5. Ejecute en linea de comandos mvn package.
6. Ejecute java -cp target/PspProgram3-1.0-SNAPSHOT.jar com.conceptosavanzados.pspprogram3.PspProgram3
7. Para realizar las pruebas unitarias ejecute en linea de comandos mvn test.


Nota: 
1. Verificar que la versión de java utilizada sea la 1.8
